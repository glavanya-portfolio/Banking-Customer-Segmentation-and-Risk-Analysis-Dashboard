# Banking Customer Analytics Project

## Project Overview

This project demonstrates an end-to-end Banking Customer Analytics solution using Python, SQL, and Power BI. The project focuses on customer demographics, account balances, loans, deposits, customer churn, and fraud analysis to generate business insights and support data-driven decision making.

## Objectives

- Generate realistic banking customer datasets using Python
- Design and implement a relational database in MySQL
- Perform data cleaning and validation
- Analyze customer demographics and behavior
- Identify customer churn risk
- Detect potential fraud indicators
- Build interactive Power BI dashboards

## Dataset

The project contains:

- Customers
- Accounts
- Transactions
- Loans
- Deposits
- Customer Churn Data

## Business Questions Answered

- Who are the highest-value customers?
- Which customers are at risk of churn?
- What are the most profitable customer segments?
- Which regions generate the highest deposits?
- How do loan amounts vary across customer groups?
- Which transactions indicate potential fraud?
